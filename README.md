Movie Ticket Booking UI (React + Tailwind CSS)

A frontend-only Movie Ticket Booking UI built using React.js and Tailwind CSS.
This project focuses on creating a modern, responsive, and user-friendly movie ticket booking interface.

Features

Browse movie listings

View movie details and show timings

Seat selection UI

Ticket booking interface (UI only)

Fully responsive design

Reusable React components

Clean and modern UI with Tailwind CSS
